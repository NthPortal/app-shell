## Add as a Dependency

### SBT
```
"com.nthportal" %% "app-shell" % "1.2.0"
```

### Maven
```
<dependency>
  <groupId>com.nthportal</groupId>
  <artifactId>app-shell_2.12</artifactId>
  <version>1.2.0</version>
</dependency>
```
